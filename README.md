# Openlearnia Markdown Editor

Browser-based Markdown workspace from [Openlearnia](https://openlearnia.com).

## Quick start

```bash
bun install
bun run dev
```

## Build & test

```bash
bun run build
bun run test
```

## Deploy

Static SPA — build `dist/` and deploy to Cloudflare Pages. Root directory `.` (repo root). Custom domain option: `markdown.openlearnia.com`.
