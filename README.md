# Markdown Editor

Browser-based Markdown workspace from [Openlearnia](https://openlearnia.com).

Source and development docs live under [`projects/tools/markdown-editor/`](projects/tools/markdown-editor/).

## Quick start

```bash
cd projects/tools/markdown-editor
bun install
bun run dev
```

## Build & test

```bash
cd projects/tools/markdown-editor
bun run build
bun run test
```

## Deploy

Static SPA — build `dist/` and deploy to Cloudflare Pages. Custom domain option: `markdown.openlearnia.com`.
